# Evaluacion Front End

Proyecto web desarrollado como evaluacion de Front End, enfocado en mostrar la historia de los campeones de Majors de Counter-Strike (CS:GO y CS2).

## Vista General

La pagina presenta:

- Una seccion hero con resumen del contenido.
- Filtros por version del juego (CS:GO y CS2).
- Una galeria dinamica de equipos campeones.
- Enlaces directos a busquedas de grandes finales en YouTube.
- Una linea de tiempo con hitos historicos de los Majors.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5 (CDN)
- Google Fonts

## Como Ejecutarlo

1. Clona o descarga este repositorio.
2. Abre el archivo `prueba3.html` en tu navegador.
3. Asegurate de mantener la carpeta `imagenes` en la ruta esperada para que se carguen los logos.
